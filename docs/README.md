# PathOwl Strategic Documents

This directory contains the three foundational documents that govern every product, engineering, sales, and design decision at PathOwl.

Last locked: May 18, 2026

## Documents

- **strategy.md** — Strategic Foundation. Who PathOwl is, who it sells to, V1 scope, trajectory, success criteria, non-goals, pricing model. The North Star. Reviewed quarterly; not edited casually.

- **demo-spec.md** — V1 Demo Specification. What gets built by August 10, 2026. The agent reasoning chain, demo scenarios, UI structure, technical architecture. Evolves as the build progresses.

- **design-language.md** — Official brand identity and design system. Light canvas, Midnight Ink (#0B1F3A), PathOwl Forest (#1F4D3A), Inter typography, Stripe/Anthropic-grade restraint. Applies to every external artifact PathOwl produces.

## Rules of engagement

Before any decision that affects scope, target customer, integration priorities, product direction, or visual identity, open these documents and ask: does this advance what we committed to, or does this expand scope? Scope expansion requires deliberate reconsideration of these documents, not a casual override.
